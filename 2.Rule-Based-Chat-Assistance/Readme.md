# 🤖 AI ChatBot (Python)

A simple yet interactive **rule-based AI ChatBot** built using **Python**. This project demonstrates the fundamentals of chatbot development, including user interaction, keyword-based response matching, personalized greetings, and conversational flow.

Designed as a beginner-friendly project, it showcases core Python concepts while maintaining clean and readable code.

---

## 📌 Project Overview

The chatbot greets the user based on the current time of day, accepts natural language input, and provides predefined responses for common questions. It continues the conversation until the user types **"bye"**.

This project is ideal for beginners learning:
- Python programming
- Functions
- Dictionaries
- Loops
- Conditional statements
- String manipulation
- Basic chatbot logic

---

## ✨ Features

- 👋 Personalized greeting using the user's name
- 🕒 Time-based greetings (Morning, Afternoon, Evening, Night)
- 💬 Interactive command-line conversation
- 🧠 Keyword-based response matching
- ⚡ Fast and lightweight implementation
- 🔁 Continuous conversation loop
- 🚪 Exit command using **"bye"**
- 🧩 Easy to extend with new responses

## 🧠 How It Works

1. Takes the user's name as input.
2. Detects the current system time.
3. Displays an appropriate greeting.
4. Stores chatbot knowledge using a Python dictionary.
5. Matches user input with predefined keywords.
6. Returns the corresponding response.
7. Ends the conversation when the user types **"bye"**.
