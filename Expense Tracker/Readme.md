
# 💰 Expense Tracker (Python Console Application)

A simple and efficient **Expense Tracker** built using **Python** that helps users record, organize, and monitor their daily expenses through an interactive command-line interface.

This project demonstrates the fundamentals of Python programming, including loops, conditional statements, lists, dictionaries, user input handling, and basic data management.

---

## 📸 Project Overview

The Expense Tracker allows users to:

- ➕ Add new expenses
- 📋 View all recorded expenses
- 💵 Calculate total expenses
- 🚪 Exit the application safely

The application stores each expense as a Python dictionary and maintains all records using a list, making it a great beginner-friendly project to understand data structures and program flow.

---

- ✅ Interactive menu-driven interface
- ✅ Add expense details

## 📖 How to Use

When you run the program, you'll see the following menu:

```
===== Expense Tracker =====

1. Add Expense
2. View All Expenses
3. View Total Expenses
4. Exit
```

### Option 1 – Add Expense

Enter:
- Expense Date
- Category
- Description
- Amount

The expense will be stored successfully.

### Option 2 – View All Expenses

Displays all recorded expenses in a readable format.

### Option 3 – View Total Expenses

Calculates and displays the total amount spent.

### Option 4 – Exit

Closes the application safely.
