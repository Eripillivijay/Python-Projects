
## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl + N | New File |
| Ctrl + O | Open File |
| Ctrl + S | Save File |
| Ctrl + E | Exit |

---

## Features Explained

### New File
Creates a blank document and resets the editor.

### Open File
Opens existing `.txt` files.

### Save File
Saves the current document using the Save dialog.

### Live Status Bar

Displays

- Word Count
- Character Count

while typing.
